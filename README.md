# 🌟 Personal Portfolio Website  

A **personal portfolio website** built using **HTML, CSS, and Boxicons** to showcase skills, projects, and contact information.  
Designed for front-end developers to demonstrate their work and online presence.

---

## 📖 Overview  
This project is a personal portfolio website highlighting:  
- **Header** with name/logo and navigation  
- **About Me** section with bio and image  
- **Education** section detailing academic journey  
- **Skills** section with coding proficiency  
- **Projects** section to showcase work samples  
- **Resume download link**  
- **Contact form**  
- **Footer** with copyright and navigation  

The website focuses on **clean design, responsiveness, and user-friendly interface**.  

---

## ✨ Features  
- 🎨 Modern and visually appealing design  
- 👩‍💻 Responsive layout for desktop and mobile  
- 🛠️ Showcases skills, education, and projects  
- 📧 Contact form for visitors  
- 📄 Resume download link  
- 🔗 Social media integration using Boxicons  

---

## 🛠️ Tech Stack  
- **HTML5** → Structure and layout  
- **CSS3** → Styling and animations  
- **Boxicons** → Icons for social links and navigation
